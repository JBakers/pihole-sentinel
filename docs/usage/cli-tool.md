# Pi-hole Sentinel CLI Tool Usage

The `pisen` CLI tool provides convenient commands for managing your Pi-hole Sentinel High Availability setup.

## Commands

- `pisen status`      - Show service status
- `pisen logs`        - Tail service logs
- `pisen vip`         - Check VIP location
- `pisen dashboard`   - Show dashboard access info
- `pisen health`      - Comprehensive health check
- `pisen test`        - Failover testing guide
- `pisen --version`   - Show version
- `pisen --help`      - Show help

## Example Usage

```bash
pisen status
pisen logs
pisen vip
pisen dashboard
pisen health
pisen test
```
